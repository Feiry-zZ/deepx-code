为什么选择 DeepX

六大核心能力

⚡

双模型智能路由

flash 起手处理机械单步，关键词命中或任务超阈值时自动升 pro。本地路由，零 LLM 调用延迟，最大化 token 经济。

🔀

Plan DAG 并行调度

LLM 主动拆解复杂任务为 DAG，deepx 按依赖关系并发执行多个子 agent，每个节点独立选择 flash 或 pro 模型。

🛡️

审核模式，安全可控

默认 review 模式：Write / Update / Bash 需人工确认。一键 /auto 切换全自动，/plan 切只读。你的代码你做主。

📷

本地 OCR，补齐视觉

Ctrl+V 粘贴图片自动调 PaddleOCR 识别文字，补齐 DeepSeek 不支持图片的最大短板。首次下载约 37MB 模型。

💬

会话压缩与持久化

长对话自动分层压缩保留关键决策；按 workspace 路径分库，对话按天落盘，重启自动续上，DeepSeek KV cache 友好。

🔧

Skills + Memory 按需加载

跨 workspace/global 多路径发现 Skills，兼容 Claude Code 生态。Memory 工具按关键词检索历史，不预注入，不占 token。

开始使用

一行命令，即刻安装

curl -fsSL https://raw.githubusercontent.com/itmisx/deepx/main/scripts/install.sh | bash
irm https://raw.githubusercontent.com/itmisx/deepx/main/scripts/install.ps1 | iex
git clone https://github.com/itmisx/deepx.git
cd deepx
go build . # 需要 Go 1.25+，Windows 产物为 deepx.exe
